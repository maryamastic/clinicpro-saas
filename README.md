# ClinicPro - Clinic Appointment SaaS

A full-stack SaaS application for clinics where patients can book appointments and doctors/admins can manage them.

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB
- JWT
- Docker
- GitHub Actions
- Render Deployment

## Features
- Patient registration/login
- Doctor dashboard
- Admin dashboard
- Appointment booking
- JWT authentication
- REST API
- Dockerized backend
- CI pipeline with GitHub Actions
- Cloud deployment

## DevOps Features
- Dockerfile
- Docker Compose
- GitHub Actions CI
- Environment variables
- Health check endpoint
- Production deployment
